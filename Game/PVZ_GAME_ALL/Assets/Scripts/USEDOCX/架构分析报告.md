### 2.2 核心组件分析

#### 2.2.1 接口层 (IPool / IPoolObject)

```csharp
// 通用对象池接口 - 简洁明了
public interface IPool<T> where T : class
{
    int Count { get; }
    int ActiveCount { get; }
    int Capacity { get; set; }
    T Get();
    void Release(T obj);
    void Clear();

```

**优点**：
- 泛型约束 `where T : class` 确保类型安全

#### 2.2.3 句柄机制 (PoolHandle)

**RAII模式应用** ([PoolHandle.cs](file:///d:\Soft\Unity\UnityProjects\PVZ_ALL\Game\PVZ_GAME_ALL\Assets\Scripts\ObjectPool\Scripts\Core\PoolHandle.cs)):

```csharp
public struct PoolHandle<T> : IDisposable
{
    public void Dispose()
    {
        if (_pool != null && _value != null)
            _pool.Release(_value);
    }
}
```

**优点**：
- using语句自动释放资源
- 防止内存泄漏
- 隐式bool转换提升体验

## 四、优点总结

### 4.1 架构亮点

1. **多类型池支持**
   - 通用对象池 (ObjectPool<T>)
   - GameObject池 (GameObjectPool)
   - 集合池 (ListPool<T>, DictionaryPool<TKey,TValue>)

2. **灵活的的生命周期管理**
   ```csharp
   // 自定义回调
   var pool = new ObjectPool<MyClass>(
       createFunc: () => new MyClass(),
       actionOnGet: obj => obj.Initialize(),
       actionOnRelease: obj => obj.Cleanup(),
       actionOnDestroy: obj => obj.Dispose()
   );
   ```

3. **PoolHandle RAII机制**
   ```csharp
   using (var handle = pool.GetHandle())
   {
       var obj = handle.Value;
       // 使用对象，自动释放
   }
   ```

4. **场景切换自动管理**
   - autoReleaseOnSceneChange 配置项
   - 避免场景切换后的资源残留

5. **静态集合池优化**
   ```csharp
   // 零GC的List使用
   using (var handle = ListPool<int>.GetHandle())
   {
       var list = handle.Value;
       // 使用后自动回收
   }
   ```

### 4.2 性能优化措施

| 优化点 | 实现方式 |
|--------|----------|
| 预分配 | Preload/WarmUp方法 |
| 栈存储 | Stack<T> 无GC分配 |
| 对象复用 | 重复利用而非销毁 |
| 延迟释放 | ReleaseToPoolDelayed |

## 六、使用案例

### 6.1 基础GameObject池使用

```csharp
public class Bullet : MonoBehaviour, IPoolObject
{
    public void OnGetFromPool() => gameObject.SetActive(true);
    public void OnReleaseToPool() { }
    public void OnDestroyFromPool() => Destroy(gameObject);
}

// 方式一：直接使用池
var pool = new GameObjectPool(bulletPrefab, transform, new PoolConfig(10, 100));
var bullet = pool.Get();
pool.Release(bullet);

// 方式二：通过Manager管理
var pool = ObjectPoolManager.Instance.GetOrCreatePool("Bullet", bulletPrefab, PoolConfig.Medium);
var bullet = pool.Get(position, rotation);
ObjectPoolManager.Instance.Despawn("Bullet", bullet);
```

### 6.2 扩展方法使用 (推荐)

```csharp
// 创建池
bulletPrefab.CreatePool("Bullet", PoolConfig.Medium);

// 获取对象
var bullet = bulletPrefab.GetFromPool("Bullet", position, rotation);
var bulletComp = bulletPrefab.GetFromPool<Bullet>(position, rotation);

// 释放对象
bullet.ReleaseToPool();
bullet.ReleaseToPoolDelayed(2.0f); // 2秒后回收
```

### 6.3 通用对象池使用

```csharp
// 创建池
var pool = ObjectPoolManager.Instance.GetOrCreatePool<MyClass>("MyClass", 
    () => new MyClass(),
    obj => obj.OnGet(),      // 获取回调
    obj => obj.OnRelease(),  // 释放回调
    config: PoolConfig.Small);

// 使用Handle自动释放
using (var handle = pool.GetHandle())
{
    var obj = handle.Value;
    obj.DoSomething();
} // 自动释放回池
```

### 6.4 List/Dictionary池使用

```csharp
// List池 - 零GC
using (var handle = ListPool<int>.GetHandle())
{
    var list = handle.Value;
    list.Add(1);
    list.Add(2);
    ProcessList(list);
} // 自动Clear并回收

// Dictionary池
using (var handle = DictionaryPool<string, GameObject>.GetHandle())
{
    var dict = handle.Value;
    dict["player"] = playerObj;
    dict["enemy"] = enemyObj;
} // 自动Clear并回收
```

### 6.5 完整游戏对象示例

```csharp
// 子弹系统
public class BulletSystem : MonoBehaviour
{
    [SerializeField] private GameObject bulletPrefab;
    
    private void Start()
    {
        bulletPrefab.CreatePool("Bullet", new PoolConfig(20, 100));
    }
    
    public void Fire(Vector3 position, Quaternion rotation)
    {
        var bullet = bulletPrefab.GetFromPool("Bullet", position, rotation);
        var bulletComp = bullet.GetComponent<Bullet>();
        bulletComp.Launch();
        
        // 3秒后自动回收
        bullet.ReleaseToPoolDelayed(3.0f);
    }
}

// 爆炸效果池
public class ExplosionEffect : MonoBehaviour, IPoolObject
{
    public void OnGetFromPool()
    {
        gameObject.SetActive(true);
        PlayEffect();
    }
    
    public void OnReleaseToPool() { }
    
    public void OnDestroyFromPool() { }
    
    private void PlayEffect()
    {
        StartCoroutine(CoPlay());
    }
    
    private IEnumerator CoPlay()
    {
        yield return new WaitForSeconds(2f);
        gameObject.ReleaseToPool();
    }
}
```
