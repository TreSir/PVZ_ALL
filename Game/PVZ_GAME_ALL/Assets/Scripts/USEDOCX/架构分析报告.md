## 四、核心系统详解

### 4.1 GameInitializer (游戏初始化器)

**职责：** 自动发现、注册、初始化所有游戏系统

**关键特性：**
- 单例模式，DontDestroyOnLoad
- 反射扫描所有程序集
- 按 Priority 排序初始化
- 支持 Update/LateUpdate 回调

```csharp
// 系统发现流程
AppDomain.CurrentDomain.GetAssemblies()
    → 扫描所有类型
    → 筛选 IGameSystem 实现
    → Activator.CreateInstance()
    → 注册到 _systems 列表
```

### 4.2 IGameSystem (系统接口)

**位置：** `GameBase/IGameSystem.cs`

```csharp
public interface IGameSystem
{
    int Priority { get; }    // 初始化优先级
    void Initialize();       // 初始化
    void Shutdown();         // 关闭
}

// 扩展接口
public interface IGameSystemPreload : IGameSystem { void Preload(); }
public interface IGameSystemUpdate : IGameSystem { void Update(float dt); }
public interface IGameSystemLateUpdate : IGameSystem { void LateUpdate(float dt); }
```

### 4.3 AudioSystem (音频系统)

**架构：** 包装器模式

```
AudioSystem (IGameSystem 实现)
     │
     └─► AudioManager (MonoBehaviour 单例)
              │
              ├─► BGM 播放
              ├─► SFX 播放
              └─► ObjectPool 管理 AudioSource
```

**配置驱动：**
```csharp
// GameSettingConfig.asset
audio:
  masterVolume: 1.0
  bgmVolume: 0.8
  sfxVolume: 1.0
  startupBGMPath: "Audio/BGM/B_001"

pool:
  audioInitialPoolSize: 10
  audioMaxPoolSize: 30
```

### 4.4 BaseUIManager (UI系统)

**架构：** 直接实现 IGameSystem

```
BaseUIManager (IGameSystem 实现)
     │
     ├─► 面板管理 (打开/关闭)
     ├─► 面板对象池
     └─► 层级管理
```

**配置驱动：**
```csharp
// GameSettingConfig.asset
ui:
  startupPanelName: "MainMenuPanel"
  enablePanelPool: true
  panelPoolInitialSize: 3
```

## 六、设计模式应用

| 模式 | 应用位置 | 说明 |
|------|----------|------|
| 单例模式 | GameInitializer, AudioManager, BaseUIManager | 全局唯一访问点 |
| 工厂模式 | ResourceManager | 资源创建抽象 |
| 策略模式 | IResourceLoader | 多种加载策略 |
| 观察者模式 | EventBus | 事件驱动通信 |
| 对象池模式 | ObjectPool, AudioManager, BaseUIManager | 复用对象减少GC |
| 接口隔离 | IGameSystem 及其扩展接口 | 职责分离 |

## 七、扩展指南

### 7.1 添加新系统

只需创建一个实现 `IGameSystem` 的类：

```csharp
using GameBase;

public class MyNewSystem : IGameSystem
{
    public int Priority => 1500;  // 在 Audio(1000) 和 UI(2000) 之间

    public void Initialize()
    {
        // 初始化逻辑
    }

    public void Shutdown()
    {
        // 清理逻辑
    }
}
```

GameInitializer 会自动发现并初始化！

### 7.2 添加新配置

1. 在 `GameSettingConfig.cs` 中添加配置类
2. 在 Unity 中更新 `GameSettingConfig.asset`
