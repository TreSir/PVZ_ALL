# UI 模块使用指南

## 概述

UI 模块负责游戏中所有界面面板的管理，包括面板的打开、关闭、层级管理和对象池化。

---

## 面板生命周期

```
None → Opening → Opened → Paused → Resumed → Closing → Closed
```

---

## 创建面板

```csharp
using UI;

public class MainMenuPanel : BasePanel
{
    public override string UIName => "MainMenuPanel";

    public override void InitUI()
    {
    }

    protected override void OnOpened()
    {
    }

    protected override void OnBeforeClose()
    {
    }
}
```

---

## 面板管理

```csharp
using UI;

BaseUIManager.Instance.OpenPanel("MainMenuPanel");
BaseUIManager.Instance.ClosePanel("MainMenuPanel");
BaseUIManager.Instance.CloseAllPanel();
```

---

## 生命周期回调

```csharp
protected virtual void OnBeforeOpen() { }
protected virtual void OnOpened() { }
protected virtual void OnPause() { }
protected virtual void OnResume() { }
protected virtual void OnBeforeClose() { }
```

---

## 配置说明

**位置：** `Resources/Configs/GameSettingConfig.asset`

| 配置项 | 说明 |
|--------|------|
| startupPanelName | 启动面板名称 |
| enablePanelPool | 是否启用面板池 |
| panelPoolInitialSize | 面板池初始大小 |

---

## 目录结构

```
Resources/
└── Prefabs/
    └── UI/
        ├── MainMenuPanel.prefab
        └── SettingsPanel.prefab
```

---

## API 速查

| 方法 | 说明 |
|------|------|
| `OpenPanel(name)` | 打开面板 |
| `ClosePanel(name)` | 关闭面板 |
| `CloseAllPanel()` | 关闭所有面板 |
| `GetPanel(name)` | 获取面板实例 |
| `SetUsePool(use, size)` | 设置池配置 |

---

## 注意事项

- 面板 Prefab 放在 `Resources/Prefabs/UI/` 目录
- 面板名称需在 `UIConst` 中定义或注册到 `_pathDict`
