# UI 模块使用指南

## 概述

UI 模块负责游戏中所有界面面板的管理，包括面板的打开、关闭、层级管理和对象池化。`BaseUIManager` 直接实现 `IGameSystem` 接口，由 `GameInitializer` 自动发现和初始化。

## 架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                    GameInitializer                          │
│                  (自动发现 IGameSystem)                      │
└─────────────────────────┬───────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│                    BaseUIManager                            │
│              (IGameSystem, Priority: 2000)                  │
│                                                             │
│  Initialize() ──► 加载配置、打开启动面板                      │
│  Shutdown()  ──► 关闭所有面板、清理池                        │
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐ │
│  │  面板管理   │  │  层级管理   │  │  对象池管理         │ │
│  │ Open/Close │  │ SortingOrder│  │ Panel Pool          │ │
│  └─────────────┘  └─────────────┘  └─────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│                      BasePanel                              │
│                    (面板基类)                                │
│                                                             │
│  生命周期: Opening → Opened → Paused → Resumed → Closing   │
│                                                             │
│  回调: OnBeforeOpen, OnOpened, OnPause, OnResume, OnClose   │
└─────────────────────────────────────────────────────────────┘
```

## 配置说明

### GameSettingConfig UI 配置

**位置：** `Resources/Configs/GameSettingConfig.asset`

```csharp
[Header("UI Settings")]
public class UISettings
{
    [Tooltip("游戏启动时打开的第一个 UI 面板名称")]
    public string startupPanelName = "MainMenuPanel";

    [Tooltip("是否启用 UI 面板对象池")]
    public bool enablePanelPool = true;

    [Tooltip("UI 面板池初始大小")]
    [Min(1)]
    public int panelPoolInitialSize = 3;
}
```

## 面板生命周期

### 生命周期状态

```csharp
public enum UILifeCycleState
{
    None,       // 初始状态
    Opening,    // 正在打开
    Opened,     // 已打开
    Paused,     // 暂停（被其他面板遮挡）
    Resumed,    // 恢复
    Closing,    // 正在关闭
    Closed      // 已关闭
}
```

### 生命周期回调

```csharp
public abstract class BasePanel : MonoBehaviour
{
    // 必须实现
    public abstract string UIName { get; }
    public abstract void InitUI();

    // 可选重写
    protected virtual void OnBeforeOpen() { }  // 打开前
    protected virtual void OnOpened() { }      // 打开后
    protected virtual void OnPause() { }       // 暂停时
    protected virtual void OnResume() { }      // 恢复时
    protected virtual void OnBeforeClose() { } // 关闭前
}
```

## 使用示例

### 1. 创建面板

```csharp
using UnityEngine;
using UnityEngine.UI;
using UI;

public class MainMenuPanel : BasePanel
{
    public override string UIName => "MainMenuPanel";

    private Button _startButton;
    private Button _settingsButton;
    private Button _quitButton;

    public override void InitUI()
    {
        _startButton = transform.Find("StartButton").GetComponent<Button>();
        _settingsButton = transform.Find("SettingsButton").GetComponent<Button>();
        _quitButton = transform.Find("QuitButton").GetComponent<Button>();

        _startButton.onClick.AddListener(OnStartClick);
        _settingsButton.onClick.AddListener(OnSettingsClick);
        _quitButton.onClick.AddListener(OnQuitClick);
    }

    protected override void OnOpened()
    {
        Debug.Log("主菜单已打开");
    }

    private void OnStartClick()
    {
        BaseUIManager.Instance.ClosePanel("MainMenuPanel");
        BaseUIManager.Instance.OpenPanel("GamePanel");
    }

    private void OnSettingsClick()
    {
        BaseUIManager.Instance.OpenPanel("SettingsPanel");
    }

    private void OnQuitClick()
    {
        #if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false;
        #else
        Application.Quit();
        #endif
    }
}
```

### 2. 带生命周期的面板

```csharp
using UnityEngine;
using UI;

public class GamePanel : BasePanel
{
    public override string UIName => "GamePanel";

    protected override void OnBeforeOpen()
    {
        Debug.Log("游戏面板即将打开");
        LoadGameData();
    }

    protected override void OnOpened()
    {
        Debug.Log("游戏面板已打开");
        StartGame();
    }

    protected override void OnPause()
    {
        Debug.Log("游戏面板暂停（被其他面板遮挡）");
        PauseGame();
    }

    protected override void OnResume()
    {
        Debug.Log("游戏面板恢复");
        ResumeGame();
    }

    protected override void OnBeforeClose()
    {
        Debug.Log("游戏面板即将关闭");
        SaveGameData();
    }
}
```

### 3. 打开/关闭面板

```csharp
using UI;

public class UIManager : MonoBehaviour
{
    public void OpenMainMenu()
    {
        var panel = BaseUIManager.Instance.OpenPanel("MainMenuPanel");
        if (panel != null)
        {
            Debug.Log("主菜单已打开");
        }
    }

    public void CloseMainMenu()
    {
        BaseUIManager.Instance.ClosePanel("MainMenuPanel");
    }

    public void CloseAllPanels()
    {
        BaseUIManager.Instance.CloseAllPanel();
    }

    public bool IsMainMenuOpen()
    {
        return BaseUIManager.Instance.IsPanelOpened("MainMenuPanel");
    }
}
```

```csharp
using UI;

public class GamePreloader : MonoBehaviour
{
    private void Start()
    {
        // 预加载常用面板
        BaseUIManager.Instance.PreloadPanel("SettingsPanel");
        BaseUIManager.Instance.PreloadPanel("PausePanel");
    }
}
```

### 清理面板池

```csharp
using UI;

public class SceneTransition : MonoBehaviour
{
    public void OnSceneChange()
    {
        BaseUIManager.Instance.CloseAllPanel();
        BaseUIManager.Instance.ClearPool();
    }
}
```

## 配置面板路径

在 `BaseUIManager.InitDics()` 中添加面板路径：

```csharp
private void InitDics()
{
    _pathDict = new Dictionary<string, string>()
    {
        {UIConst.MainMenuPanel, "MainMenuPanel"},
        {UIConst.GamePanel, "GamePanel"},
        {UIConst.SettingsPanel, "SettingsPanel"},
        {UIConst.PausePanel, "PausePanel"}
    };
}

public class UIConst
{
    public const string MainMenuPanel = "MainMenuPanel";
    public const string GamePanel = "GamePanel";
    public const string SettingsPanel = "SettingsPanel";
    public const string PausePanel = "PausePanel";
}
```

## 注意事项

1. **UIName 唯一性**：每个面板的 UIName 必须唯一
2. **路径配置**：面板路径必须在 `_pathDict` 中配置
3. **BasePanel 组件**：面板 Prefab 必须挂载继承自 BasePanel 的脚本
4. **生命周期**：在适当的回调中处理数据加载和保存
5. **对象池**：频繁打开关闭的面板建议启用池化
6. **空值检查**：面板可能为 null，使用前检查
