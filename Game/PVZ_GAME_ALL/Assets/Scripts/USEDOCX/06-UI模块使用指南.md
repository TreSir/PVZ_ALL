# UI 模块使用指南

## 概述

UI 模块提供一套完整的游戏 UI 管理系统，支持面板的生命周期管理、对象池化、层级管理等特性。通过 BasePanel 和 BaseUIManager 的配合，可以高效管理游戏中的各种界面。

## 核心组件

### BasePanel - 面板基类

所有 UI 面板需要继承 BasePanel：

```csharp
public abstract class BasePanel : MonoBehaviour
{
    // 面板名称（唯一标识）
    public abstract string UIName { get; }

    // 初始化 UI 组件
    public abstract void InitUI();
}
```

### BaseUIManager - 面板管理器

负责面板的打开、关闭、层级管理：

```csharp
// 打开面板
BasePanel panel = BaseUIManager.Instance.OpenPanel("PanelName");

// 关闭面板
BaseUIManager.Instance.ClosePanel("PanelName");

// 关闭所有面板
BaseUIManager.Instance.CloseAllPanel();
```

### UILifeCycleState - 生命周期状态

```csharp
public enum UILifeCycleState
{
    None,       // 初始状态
    Opening,    // 正在打开
    Opened,     // 已打开
    Paused,     // 暂停（被其他面板遮挡）
    Resumed,    // 恢复
    Closing,    // 正在关闭
    Closed      // 已关闭
}
```

## 基础使用

### 1. 创建面板

创建一个继承自 BasePanel 的脚本：

```csharp
using UnityEngine;
using UI;

public class MainMenuPanel : BasePanel
{
    // 必须实现：面板唯一名称
    public override string UIName => "MainMenuPanel";

    // 可以添加 UI 组件引用
    private UnityEngine.UI.Button _startButton;
    private UnityEngine.UI.Button _settingsButton;
    private UnityEngine.UI.Text _titleText;

    // 初始化 UI 组件
    public override void InitUI()
    {
        // 获取子对象
        _startButton = transform.Find("StartButton").GetComponent<UnityEngine.UI.Button>();
        _settingsButton = transform.Find("SettingsButton").GetComponent<UnityEngine.UI.Button>();
        _titleText = transform.Find("Title").GetComponent<UnityEngine.UI.Text>();

        // 添加点击事件
        _startButton.onClick.AddListener(OnStartClick);
        _settingsButton.onClick.AddListener(OnSettingsClick);
    }

    private void OnStartClick()
    {
        Debug.Log("点击了开始按钮");
    }

    private void OnSettingsClick()
    {
        Debug.Log("点击了设置按钮");
    }
}
```

### 2. 配置面板路径

在 BaseUIManager 中配置面板路径：

```csharp
// 在 BaseUIManager.InitDics() 中添加
_pathDict = new Dictionary<string, string>()
{
    {UIConst.MainMenuPanel, "MainMenuPanel"},
    {UIConst.SettingsPanel, "SettingsPanel"},
    {UIConst.GamePanel, "GamePanel"}
};
```

### 3. 打开/关闭面板

```csharp
// 打开面板
var panel = BaseUIManager.Instance.OpenPanel("MainMenuPanel");

// 关闭面板（放入对象池）
BaseUIManager.Instance.ClosePanel("MainMenuPanel");

// 关闭面板（不放入对象池，直接销毁）
BaseUIManager.Instance.ClosePanel("MainMenuPanel", wantRemove: false);

// 关闭所有面板
BaseUIManager.Instance.CloseAllPanel();
```

## 生命周期回调

### 可用的回调方法

```csharp
protected virtual void OnBeforeOpen()   // 打开前调用
protected virtual void OnOpened()       // 打开后调用
protected virtual void OnPause()        // 暂停时调用（被其他面板遮挡）
protected virtual void OnResume()       // 恢复时调用
protected virtual void OnBeforeClose() // 关闭前调用
```

### 使用示例

```csharp
public class GamePanel : BasePanel
{
    public override string UIName => "GamePanel";

    public override void InitUI() { }

    // 打开前：播放动画、加载数据
    protected override void OnBeforeOpen()
    {
        Debug.Log("游戏面板即将打开");
        LoadGameData();
    }

    // 打开后：开始游戏逻辑
    protected override void OnOpened()
    {
        Debug.Log("游戏面板已打开");
        StartGame();
    }

    // 暂停：暂停游戏
    protected override void OnPause()
    {
        Debug.Log("游戏面板暂停");
        GamePause();
    }

    // 恢复：恢复游戏
    protected override void OnResume()
    {
        Debug.Log("游戏面板恢复");
        GameResume();
    }

    // 关闭前：保存数据
    protected override void OnBeforeClose()
    {
        Debug.Log("游戏面板即将关闭");
        SaveGameData();
    }
}
```

## 对象池化

### 启用面板池化

```csharp
// 在游戏开始时调用
BaseUIManager.Instance.SetUsePool(true, initialPoolSize: 5);
```

### 预加载面板

```csharp
// 预加载常用面板
BaseUIManager.Instance.PreloadPanel("MainMenuPanel");
BaseUIManager.Instance.PreloadPanel("SettingsPanel");
```

### 清除池

```csharp
// 清理所有面板池（切换存档时使用）

```

## 完整示例

### 示例1：主菜单

```csharp
using UnityEngine;
using UI;

public class MainMenuPanel : BasePanel
{
    public override string UIName => "MainMenuPanel";

    private UnityEngine.UI.Button _startButton;
    private UnityEngine.UI.Button _settingsButton;
    private UnityEngine.UI.Button _quitButton;

    public override void InitUI()
    {
        _startButton = transform.Find("StartButton").GetComponent<UnityEngine.UI.Button>();
        _settingsButton = transform.Find("SettingsButton").GetComponent<UnityEngine.UI.Button>();
        _quitButton = transform.Find("QuitButton").GetComponent<UnityEngine.UI.Button>();

        _startButton.onClick.AddListener(OnStartClick);
        _settingsButton.onClick.AddListener(OnSettingsClick);
        _quitButton.onClick.AddListener(OnQuitClick);
    }

    private void OnStartClick()
    {
        // 打开游戏面板
        BaseUIManager.Instance.OpenPanel("GamePanel");
        // 关闭自己
        BaseUIManager.Instance.ClosePanel("MainMenuPanel");
    }

    private void OnSettingsClick()
    {
        BaseUIManager.Instance.OpenPanel("SettingsPanel");
    }

    private void OnQuitClick()
    {
#if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false;
#else
        Application.Quit();
#endif
    }

    protected override void OnOpened()
    {
        Debug.Log("主菜单已显示");
    }
}
```

### 示例2：设置面板

```csharp
using UnityEngine;
using UnityEngine.UI;
using UI;

public class SettingsPanel : BasePanel
{
    public override string UIName => "SettingsPanel";

    private Slider _bgmVolumeSlider;
    private Slider _sfxVolumeSlider;
    private Toggle _fullscreenToggle;
    private Button _closeButton;

    public override void InitUI()
    {
        _bgmVolumeSlider = transform.Find("BGMVolume").GetComponent<Slider>();
        _sfxVolumeSlider = transform.Find("SFXVolume").GetComponent<Slider>();
        _fullscreenToggle = transform.Find("Fullscreen").GetComponent<Toggle>();
        _closeButton = transform.Find("CloseButton").GetComponent<Button>();

        _bgmVolumeSlider.onValueChanged.AddListener(OnBGMVolumeChanged);
        _sfxVolumeSlider.onValueChanged.AddListener(OnSFXVolumeChanged);
        _fullscreenToggle.onValueChanged.AddListener(OnFullscreenChanged);
        _closeButton.onClick.AddListener(OnCloseClick);

        LoadCurrentSettings();
    }

    private void LoadCurrentSettings()
    {
        _bgmVolumeSlider.value = AudioManager.Instance.BGMVolume;
        _sfxVolumeSlider.value = AudioManager.Instance.SFXVolume;
        _fullscreenToggle.isOn = Screen.fullScreen;
    }

    private void OnBGMVolumeChanged(float value)
    {
        AudioManager.Instance.BGMVolume = value;
    }

    private void OnSFXVolumeChanged(float value)
    {
        AudioManager.Instance.SFXVolume = value;
    }

    private void OnFullscreenChanged(bool isFullscreen)
    {
        Screen.fullScreen = isFullscreen;
    }

    private void OnCloseClick()
    {
        BaseUIManager.Instance.ClosePanel("SettingsPanel");
    }
}
```

### 示例3：提示弹窗

```csharp
using UnityEngine;
using UI;

public class MessageBoxPanel : BasePanel
{
    public override string UIName => "MessageBoxPanel";

    private Text _messageText;
    private Button _confirmButton;
    private Button _cancelButton;

    private System.Action _onConfirm;
    private System.Action _onCancel;

    public override void InitUI()
    {
        _messageText = transform.Find("MessageText").GetComponent<Text>();
        _confirmButton = transform.Find("ConfirmButton").GetComponent<Button>();
        _cancelButton = transform.Find("CancelButton").GetComponent<Button>();

        _confirmButton.onClick.AddListener(OnConfirm);
        _cancelButton.onClick.AddListener(OnCancel);
    }

    public void SetMessage(string message, System.Action onConfirm = null, System.Action onCancel = null)
    {
        _messageText.text = message;
        _onConfirm = onConfirm;
        _onCancel = onCancel;
    }

    private void OnConfirm()
    {
        _onConfirm?.Invoke();
        BaseUIManager.Instance.ClosePanel("MessageBoxPanel");
    }

    private void OnCancel()
    {
        _onCancel?.Invoke();
        BaseUIManager.Instance.ClosePanel("MessageBoxPanel");
    }

    // 静态方法创建提示框
    public static void Show(string message, System.Action onConfirm = null)
    {
        var panel = BaseUIManager.Instance.OpenPanel("MessageBoxPanel") as MessageBoxPanel;
        panel.SetMessage(message, onConfirm);
    }
}
```

### 示例4：面板切换动画

```csharp
using System.Collections;
using UnityEngine;
using UI;

public class AnimatedPanel : BasePanel
{
    public override string UIName => "AnimatedPanel";

    public override void InitUI() { }

    protected override void OnBeforeOpen()
    {
        // 设置初始状态
        transform.localScale = Vector3.zero;
    }

    protected override void OnOpened()
    {
        // 播放打开动画
        StartCoroutine(OpenAnimation());
    }

    protected override void OnBeforeClose()
    {
        // 播放关闭动画
        StartCoroutine(CloseAnimation());
    }

    private IEnumerator OpenAnimation()
    {
        float duration = 0.3f;
        float elapsed = 0f;

        while (elapsed < duration)
        {
            elapsed += Time.deltaTime;
            float t = elapsed / duration;
            // 弹性效果
            t = Mathf.Sin(t * Mathf.PI * 0.5f);
            transform.localScale = Vector3.Lerp(Vector3.zero, Vector3.one, t);
            yield return null;
        }

        transform.localScale = Vector3.one;
    }

    private IEnumerator CloseAnimation()
    {
        float duration = 0.2f;
        float elapsed = 0f;
        Vector3 startScale = transform.localScale;

        while (elapsed < duration)
        {
            elapsed += Time.deltaTime;
            float t = elapsed / duration;
            transform.localScale = Vector3.Lerp(startScale, Vector3.zero, t);
            yield return null;
        }
    }
}
```

## 层级管理

面板会自动管理 Canvas 层级：

```csharp
// 打开面板时，sortingOrder 会自动递增
BaseUIManager.Instance.OpenPanel("Panel1"); // sortingOrder = 1
BaseUIManager.Instance.OpenPanel("Panel2"); // sortingOrder = 2
BaseUIManager.Instance.OpenPanel("Panel3"); // sortingOrder = 3
```

## 注意事项

1. **UIName 唯一性**：每个面板的 UIName 必须唯一
2. **路径配置**：面板路径必须在 pathDict 中配置
3. **BasePanel 组件**：面板 Prefab 必须挂载继承自 BasePanel 的脚本
4. **生命周期**：在适当的回调中处理数据加载和保存
5. **对象池**：频繁打开关闭的面板建议启用池化
6. **事件订阅**：面板内使用 EventBus 时，在 OnEnable/OnDisable 中订阅/取消订阅
