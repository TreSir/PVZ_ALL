### OPT-002：SourceLoad扩展

**问题位置：** [ResourceManager.cs](../SourceLoad/ResourceManager.cs)

**当前状态：** 仅实现了 ResourcesLoader

**优化方案：**
1. 创建 AddressablesLoader 实现 IResourceLoader
2. 添加加载器切换配置
3. 支持异步加载进度回调

```csharp
public class AddressablesLoader : IResourceLoader
{
    public T Load<T>(string path) where T : Object
    {
        var handle = Addressables.LoadAssetAsync<T>(path);
        return handle.WaitForCompletion();
    }

    public void LoadAsync<T>(string path, Action<T> onComplete) where T : Object
    {
        Addressables.LoadAssetAsync<T>(path).Completed += handle =>
        {
            onComplete?.Invoke(handle.Result);
        };
    }

    public void Unload(Object asset)
    {
        Addressables.Release(asset);
    }
}
```

### OPT-003：存档版本管理

**问题位置：** [SaveManager.cs](../DataManager/SaveManager.cs)

**当前状态：** 存档数据无版本号，游戏更新后无法迁移旧存档

**优化方案：**
```csharp
public interface ISaveData
{
    string lastSaveTime { get; set; }
    float playTime { get; set; }
    int version { get; set; }  // 新增版本号
}

public class SaveMigration
{
    public static void Migrate(ISaveData data, int targetVersion)
    {
        while (data.version < targetVersion)
        {
            data.version++;
            switch (data.version)
            {
                case 1: MigrateToV1(data); break;
                case 2: MigrateToV2(data); break;
            }
        }
    }
}
```

### OPT-005：资源缓存

**问题位置：** [ResourceManager.cs](../SourceLoad/ResourceManager.cs)

**当前状态：** 每次加载都调用 Resources.Load，无缓存

**优化方案：**
```csharp
private static Dictionary<string, Object> _cache = new Dictionary<string, Object>();

public static T Load<T>(string path) where T : Object
{
    if (_cache.TryGetValue(path, out var cached))
    {
        return cached as T;
    }

    var asset = _loader.Load<T>(path);
    if (asset != null)
    {
        _cache[path] = asset;
    }
    return asset;
}
```
